Retrieves tweets based on a search. Without authentication.

### Start Configuration :

- Urls : `https://twitter.com/search`
- GET Data : `q = "my search"`

Replace `my search` with your research.

### Result :

- user_id
- fullname
- badge
- username
- tweet_id
- tweet_datetime
- tweet_text
- tweet_lang
- tweet_url
- tweet_img
- tweet_reply
- tweet_retweet
- tweet_like
